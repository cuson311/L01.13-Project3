# L01.13-Project3
 
## 1. Fetch source code
```
git init
git remote add origin https://github.com/cuson311/L01.13-Project3.git
git fetch
git checkout cuson
```
## 2. Install selenium and pandas
```
pip install selenium pandas
``` 
## 3. Change path to the folder you want to use
```
cd Level_1
cd [your-folder]
```
## 4. Run code
```
python3 + [name-of-file.py]
```
